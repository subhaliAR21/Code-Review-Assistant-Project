## ğŸš€ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/codemate-ai.git
   cd codemate-ai
   ```

2. **Install client dependencies**
   ```bash
   cd client
   npm install
   ```

3. **Install server dependencies**
   ```bash
   cd ../server
   npm install
   ```

4. **Start the development servers**

   **Client (Frontend):**
   ```bash
   cd client
   npm run dev
   ```

   **Server (Backend):**
   ```bash
   cd server
   npm start
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.
